# Financial-Credit-Risk-ETL-Data-Warehousing-Pipeline
Enterprise-grade ETL pipeline processing 2.2M+ loan records using Python, Bash, and Oracle XE. Features a "Reject Link" validation layer for data quality, automated PL/SQL Star Schema transformations, and comprehensive audit logging. Optimized for high-volume ingestion with memory-efficient chunking. Perfect for demonstrating DataStage skills.
